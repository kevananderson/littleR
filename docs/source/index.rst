LittleR 
=======
**Requirements**, with configuration management by git.
-------------------------------------------------------
LittleR is a full suite for requirements management. It is built for the 
most demanding software projects. Configuration managment is provided by 
git, makeing the requirements work-flow the same as the software
development one.

.. note::
   This project is under development.
